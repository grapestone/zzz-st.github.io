2020年的第一篇博客

使用hexo搭建博客并部署到GitHub
1.下载安装node.js(下载地址：http://nodejs.org)
– 命令node -v和npm -v 验证是否安装成功
2.安装hexo

sudo npm install -g hexo-cli

– -g选项指全局安装
– hexo -v 命令验证安装结果
3. 新建目录blog，然后进入

sudo hexo init 初始化

4.启动hexo

sudo hexo s

5.部署到github上

sudo npm install –save hexo-deployer-git

6.设置blog文件夹下的_config.yum
最后两行设置为：type: git
repo: github地址
branch: master
7.部署

sudo hexo d
